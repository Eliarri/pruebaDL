Pasos para Iniciar un Repositorio Local y Subirlo a GitHub

Paso Descripción
0 Configura tu nombre de usuario y correo electrónico localmente con los comandos:
  • git config --global user.name "TuNombre"
  • git config --global user.email "tu@email.com"
1 Abre la ventana de comandos en el directorio donde deseas crear el repositorio.
2 Ejecuta el comando git init para inicializar un nuevo repositorio Git local.
3 Añade archivos al repositorio usando git add <nombre_archivo> o git add . para agregar todos los archivos.
4 Realiza el primer commit con el comando git commit -m "Primer commit".
5 Crea un nuevo repositorio en GitHub. Puedes hacerlo desde la interfaz web de GitHub.
6 Obtén la URL del repositorio remoto en GitHub.
7 Agrega el repositorio remoto a tu repositorio local con el comando git remote add origin <URL>.
8 Sube tus cambios al repositorio remoto en GitHub usando git push -u origin master (o el nombre de tu rama).
