Atajos Útiles de Visual Studio Code

Atajo Descripción
Ctrl + ` Abre o cierra la terminal integrada.
Ctrl + P Abre el cuadro de búsqueda de archivos.
Ctrl + , Abre la configuración del usuario.
Ctrl + B Activa o desactiva la barra lateral.
Ctrl + D Selecciona la próxima instancia de la palabra actual.
Ctrl + Shift + L Selecciona todas las instancias de la palabra actual.
Alt + Click Inserta múltiples cursores.
Ctrl + Alt + Up/Down Mueve la línea actual hacia arriba o hacia abajo.
Ctrl + / Comenta o descomenta la línea o el bloque seleccionado.
Ctrl + Shift + K Elimina la línea actual.
Ctrl + Shift + I Formato del documento.
Ctrl + Shift + P Abre el cuadro de comandos.
Ctrl + F Buscar en el archivo.
Ctrl + H Reemplazar en el archivo.
Ctrl + K, Ctrl + S Guarda todos los archivos.
